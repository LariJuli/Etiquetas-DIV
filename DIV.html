Contenido 1 de la etiqueta div HTML

La etiqueta Div se utiliza para agrupar muchas etiquetas HTML como tablas, listas y párrafos en una página web. La etiqueta div se utiliza principalmente en páginas web vacías para dividir o crear secciones de contenido o etiquetas HTML.

Contenido 2 de la etiqueta div HTML

Para hacer una sección de datos o contenidos requeridos se utilizan múltiples etiquetas div en el contenedor. Ejemplo: header, footer, barra de navegación, tabla, imágenes, textos, etc.
